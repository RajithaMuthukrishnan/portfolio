Object Orientation Detection Using Transfer Learning

Developed a deep learning model leveraging Siamese Neural Networks and MobileNetV2 transfer learning to predict object orientation differences with high accuracy.

Object Orientation Detection Using Transfer Learning

Project Type: Research: Model Development

Tech Stack

  • Python
  • Scikit-Image
  • TensorFlow/Keras
  • Siamese Neural Networks
  • MobileNetV2 (Transfer Learning)
  • Tum-LCP dataset

Core Features

  • Utilized Tum-LCP dataset for training and evaluation.
  • Engineered a Siamese architecture to compare object pairs and predict angular differences.
  • Integrated transfer learning (MobileNetV2) for efficient feature extraction.

Impacts/Results

  • Achieved a Mean Squared Error (MSE) of 0.0129, indicating high precision in angle prediction.
  • Demonstrated scalable approach for tasks requiring fine-grained orientation detection in images.
